# ziagakist

Personal portfolio — an interactive 3D scene built with Three.js and Vite.

## Tech

- **Three.js** — 3D scene and camera navigation
- **Vite** — build tooling
- **Vanilla JS** — no framework

## Deploy

Hosted on Vercel. Push to `main` triggers a deploy automatically.
